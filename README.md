# Gutenberg CodePen Embed

## Description
Creates a [WordPress Gutenberg](https://wordpress.org/gutenberg/) block for embedding Pens from [CodePen](http://codepen.io/).

Maintained by [Andrew Taylor](https://www.ataylor.me/).
